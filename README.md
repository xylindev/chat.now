# **Chat.Now**

## **Présentation du Projet**

**Chat.Now** est un projet personnel de petite envergure conçu dans le but d’approfondir mes connaissances en **développement back-end**. Il s’inscrit dans la continuité de mon apprentissage et me permet de mettre en pratique des concepts fondamentaux à travers une application concrète et fonctionnelle.

## **Objectif Pédagogique**

L’objectif principal de ce projet est de comprendre et maîtriser le fonctionnement d’un serveur back-end basé sur **Tomcat**.
À travers le développement de Chat.Now, je travaille notamment sur :

* la gestion des requêtes HTTP ;
* les sessions utilisateurs ;
* la communication entre le client et le serveur ;
* la logique métier côté serveur.

Ce projet me sert avant tout de terrain d’expérimentation pour consolider mes bases et identifier les bonnes pratiques à adopter.

## **Un Projet d’Apprentissage**

Chat.Now n’a pas vocation à être une application prête pour la production. Il s’agit avant tout d’un projet **éducatif**, pensé pour apprendre, tester, faire des erreurs et améliorer progressivement le code.
La sécurité, l’optimisation avancée et la montée en charge ne sont donc pas les priorités principales à ce stade.
